## About the Website

- **Purpose:** Supports my Twitch stream and RPG campaign, serving as the official online hub.
- **Features:** 
  - A campaign timeline
  - Live stream schedule
  - Detailed character pages
  - Rollable character sheets similar in interactivity to D&D Beyond
- **Design and Content:** Tailored specifically for the LoreRealm campaign and stream, with all content, layout, and integrations reflecting my original vision and programming.
